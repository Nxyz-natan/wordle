# Wordle

A Wordle clone made with html css and js 

## Features

- **Keyboard** - if you dont have a keyboard or want to use mouse only you can use the built in keyboard
- **diffuclities** - there is 2 diffuclties easy or hard easy you can guess up to 6 and hard up to 4

## How to Play

1. Guess a 5 letter word
2. see if you get green yellow or grey
3. green = right positiona and right word
4. yellow = wrong position and right right word
5. grey = neither right position or word
6. keep guessing until you get the word
